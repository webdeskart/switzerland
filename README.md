# Switzerland Travel Guide for Indian Travelers 🇨🇭

A comprehensive, professional travel guide website for Indian travelers planning a trip to Switzerland. Features 17 detailed chapters covering visas, flights, budgets, itineraries, destinations, food, shopping, and more.

## 🛠️ Features

- Multi-page website with sidebar navigation
- 32 high-quality copyright-free images
- Mobile-responsive design
- All costs in INR (₹) and CHF
- Verified 2026 pricing and information
- Print-optimized CSS
- DOCX export compatible with MS Office 2016
